How to use this Lab Tool? (كيف تستخدم الأداة؟)

01. المستمع (Listener)

في جهازك (المهاجم)، يجب فتح منفذ لاستقبال الاتصال القادم من الضحية باستخدام أداة Netcat.

nc -lvnp 4444
02. الحمولة (Payload)

أدخل الـ IP الخاص بك (الذي ستستقبل عليه الاتصال) ورقم المنفذ أدناه، ثم اختر الكود المناسب لنظام الضحية وقم بنسخه.

Target Config
03. التنفيذ (Execution)

قم بتنفيذ الكود المنسوخ في جهاز الضحية (الـ VM في المختبر). بمجرد التشغيل، ستلاحظ استلامك للـ Shell في نافذة الـ Netcat لديك.

* للاستخدام الأكاديمي فقط

TARGET CONFIG
